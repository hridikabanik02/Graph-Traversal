This program will read input from two text files - input.txt and query.txt. And then the code will be run and will output two text files called output1.txt and output2.txt
The first file will contain the paths printed after using the Depth-First traversal technique and the second file will contain the paths printed after using the Breadth-First traversal technique.
Compile by: javac Assign4.java Graph.java Queue.java
Run the program by: java Assign4 input query output1 output2
